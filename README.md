## 🚀 Featured Project

### 🎓 Predicting Student Success in Mathematics
A machine learning project that predicts whether students will pass or fail math courses.

🔍 Key insight:  
There is a tradeoff between **accuracy and early intervention**—the most accurate models aren’t always the most useful.

📊 Models Used:
- Logistic Regression  
- Decision Trees  

📈 Results:
- Best model accuracy: 88%  
- Early-warning model: ~73%
